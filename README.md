# HR Analytics Dashboard
## Tools Used: MySQL | Power BI

## Project Overview
Built an end-to-end HR Analytics Dashboard analyzing 
2,750 employee records to uncover key workforce insights 
and support data-driven HR decision making.

## Dataset
- 2,750 employee records
- 17 columns including department, salary, attrition, 
  performance rating, age, city and more

## Key Insights
- Attrition rate of 24.5% — 63% above industry benchmark
- Employees who left earned lower average salary 
  than those who stayed
- Younger employees (Under 25 and 25-34) showed 
  highest attrition rate

## SQL Analysis (MySQL)
- Created database and table from scratch
- Wrote 15+ queries covering attrition rate, 
  salary benchmarking, gender distribution, 
  hiring trends and more

## Power BI Dashboard
- 4 interactive pages
- 5 DAX measures
- 10+ visuals with dynamic slicers

## Dashboard Pages
1. Overview — Headcount, attrition rate, avg salary
2. Attrition Analysis — By department, age, salary
3. Salary & Performance — Benchmarking by department
4. Workforce Profile — Hiring trends, experience, cities
